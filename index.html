This Website

I've had half a portfolio website on the backburner of my solid state drive for years now, but with my Master's finished it's time I got it publishable so I can showcase my work. What you're looking at now is a minimal viable product.

This site's built using Gatsby. To add new portfolio items, I just add a new Markdown file to the data folder of my website. React and GraphQL handle the rest.

DIY Fume Hood

In the midst of the pandemic, I became very concerned with the potential respiratory side effects of the fumes coming off my 3D printer, for obvious reasons. To mitigate them, I built a fume hood out of the cheapest bathroom fan I could find at Home Depot, a dryer hose, and some overpriced lumber and plastic.

The 3D printer itself is a DIY hackjob based on the popular and open-source Prusa i3 MK2.

I might build another fume hood later, for one of my bestest of friends.

Project SuperLIMBinal

Developed for my capstone project at the University of Windsor with Claudia Lutfallah in 2018, Project SuperLIMBinal is a system to detect gestures using muscle activation signals and transmit them wirelessly to a robotic arm.

After being unimpressed with the widely available single-channel MyoWare sensor, we built our own with more channels for less money. The decoding of gestures works using sparse nonnegative matrix factorization, which we learned about for the purposes of gesture detection from a very interesting paper. I did the software portions of this project, and some of the electronics prototyping and development. It was pretty neat.

You can check out the codebase for this project here, on GitHub.
